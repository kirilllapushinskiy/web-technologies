Python поддерживает динамическую типизацию, то есть тип переменной определяется только во время исполнения. Поэтому вместо «присваивания значения переменной» лучше говорить о «связывании значения с некоторым именем». К примитивным типам в Python относятся булевый, целое число произвольной точности, число с плавающей запятой и комплексное число. Из контейнерных типов в Python встроены: строка, список, кортеж, словарь и множество.
Описание Пример
Тип Изменяемость
set Изменяемый {4.0, 'string', True}
dict {'key1': 1.0, 3: False}
list [4.0, 'string', True]
ellipsis Неизменяемый ...
float 1.414
int 42
bool True False
NoneType None
range range(10, -5, -2)
str "Hello, World!"
tuple (4.0, 'string', True)
Все значения являются объектами, в том числе функции, методы, модули, классы. Добавить новый тип можно либо написав класс (class), либо определив новый тип в модуле расширения (например, написанном на языке C). Система классов поддерживает наследование (одиночное и множественное) и метапрограммирование. Возможно наследование от большинства встроенных типов и типов расширений.
